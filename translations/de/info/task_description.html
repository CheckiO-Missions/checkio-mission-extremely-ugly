Hässliche Zahlen sind Zahlen, deren einzige Primfaktoren 2, 3 oder 5 sind. Die Folge 1, 2, 3, 4, 5, 6, 8, 9, 10, 12, 15, ... zeigt die ersten 11 hässlichen Zahlen. Gemäß der Konvention ist die 1 enthalten. Schreiben Sie ein Programm, das die N-te hässliche Zahl findet und ausgibt. Wie bei der vorigen Aufgabe, mit einem kleinen Unterschied - der Eingabewert ist auf 1000000 begrenzt, und die Ausgabe ist str.

Eingabe: N, int.

Ausgabe: N'th Ugly Number, str.

Beispiel:

{% if is_js %} 
extremely_ugly(4) == '4'
extremely_ugly(6) == '6'
extremely_ugly(11) == '15'
extremely_ugly(999) == '51018336'
{% else %} 
extremely_ugly(4) == '4'
extremely_ugly(6) == '6'
extremely_ugly(11) == '15'
extremely_ugly(999) == '51018336'
{% endif %}

Wie es verwendet wird: Demonstrieren, dass der direkte Weg sehr langsam sein kann

Vorbedingung: Die gegebene Zahl ist kleiner oder gleich 1000000
