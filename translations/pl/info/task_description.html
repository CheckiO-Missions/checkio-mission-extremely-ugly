Brzydkie liczby to liczby, których jedynymi czynnikami pierwszymi są 2, 3 lub 5. Ciąg 1, 2, 3, 4, 5, 6, 8, 9, 10, 12, 15, ... przedstawia 11 pierwszych liczb brzydkich. Zgodnie z konwencją, 1 jest uwzględniona. Proszę napisać program, który znajdzie i wypisze N-tą brzydką liczbę. To samo co poprzednie zadanie z drobną różnicą - limit wartości wejściowej to 1000000, a wyjście to str.

Wejście: N, int.

Wyjście: N'th Ugly Number, str.

Przykład:

{% if is_js %} 
extremely_ugly(4) == '4'
extremely_ugly(6) == '6'
extremely_ugly(11) == '15'
extremely_ugly(999) == '51018336'
{% else %} 
extremely_ugly(4) == '4'
extremely_ugly(6) == '6'
extremely_ugly(11) == '15'
extremely_ugly(999) == '51018336'
{% endif %}

Jak to jest używane: Proszę zademonstrować, że droga prosta może być bardzo wolna

Warunek wstępny: Podana liczba int jest mniejsza lub równa 1000000

Pomysł na misję został zaczerpnięty z wyzwania ICPC
