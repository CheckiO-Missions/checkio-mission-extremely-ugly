Les nombres laids sont des nombres dont les seuls facteurs premiers sont 2, 3 ou 5. La séquence 1, 2, 3, 4, 5, 6, 8, 9, 10, 12, 15, ... montre les 11 premiers nombres laids. Par convention, le 1 est inclus. Ecrivez un programme pour trouver et imprimer le N'ième nombre moche. Identique à la tâche précédente avec une différence mineure - la valeur limite d'entrée est 1000000, et la sortie est str.

Entrée: N, int.

Sortie: N'th Ugly Number, str.

Exemple:

extremely_ugly(6) == '6'
extremely_ugly(11) == '15'
extremely_ugly(999) == '51018336'
{% else %} 
extremely_ugly(4) == '4'
extremely_ugly(6) == '6'
extremely_ugly(11) == '15'
extremely_ugly(999) == '51018336'
{% endif %}

Comment c'est utilisé: Démontrer que la voie directe peut être très lente

Condition préalable: L'int donné est inférieur ou égal à 1000000
